# Medicare Healthcare Analytics Data Warehouse

## Overview

This project was developed as part of a Business Intelligence and Data Warehousing course during my Master's studies.

The objective was to design and implement a healthcare analytics data warehouse using Medicare datasets containing over one million healthcare records. The project focused on data modeling, dimensional design, OLAP cube development, and multidimensional analysis to support healthcare decision-making.

## Dataset

The project utilized Medicare healthcare data containing information related to:

- Healthcare providers
- Medical services
- Provider credentials
- Geographic regions
- Healthcare service utilization
- Medicare claims data

Dataset size:

- Over 1 million records

---

## Project Objectives

- Normalize and organize healthcare datasets
- Design dimensional data models
- Create fact and dimension tables
- Build hierarchies for multidimensional analysis
- Develop OLAP cubes using SSAS
- Support analytical reporting and business intelligence processes
